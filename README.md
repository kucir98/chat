# Chat project
## General info
Prosty chat room napisany w node.js, posiada pokoje w których użytkownicy mogą przesyłać między sobą wiadomości.
Czat nie posiada póki co systemu logowania, ani zapisywania wiadomości. Użytkonwicy posługują się nazwą podaną podczas dołączania do pokoju.
Deployement czatu został przeprowadzony na darmowym planie usługi App Service przy pomocy narzędzi dostępnych w VS Code.

	
## Setup
Można uruchomić stronę z czatem na chmurze pod adresem https://czat.azurewebsites.net/

Żeby uruchomić projekt lokalnie trzeba wykonnać poniższe komendy w folderze aplikacji:
```
$ npm install
$ npm start
```
